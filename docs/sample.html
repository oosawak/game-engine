Quick Start

1. Cube を作る

最小のオブジェクトを追加して、Scene 上で選択・移動できる状態にします。

const cube = app.spawn("Cube", {
  position: { x: 0, y: 1, z: 0 },
  size: { x: 1, y: 1, z: 1 },
});

2. Camera を追尾する

選択中のオブジェクトを Camera が見るようにして、編集しやすくします。

app.setCameraTarget(cube);
app.setCameraTarget(cube, "SceneCamera");

3. PNG を動かす

画像アニメーションをスクリプトから再生し、表示の確認を行います。

const sheet = await app.loadImageAnimation(
  ["walk_01.png", "walk_02.png"],
  { loop: true }
);
app.playImageAnimation(sheet, cube.id);

4. VRM モーション

VRM とモーションを読み込み、再生・停止・シークを操作できます。

const vrm = await app.loadVrm("assets/character.vrm");
app.playVrmMotion(vrm, "idle");
app.seekVrmMotion(vrm, 0.5);

Script Example

Editor の Play 時に流す想定の例です。オブジェクト生成、カメラ追尾、アニメーション再生をまとめて書けます。 

const cube = app.spawn("Cube", {
  position: { x: 0, y: 0, z: 0 },
  size: { x: 1, y: 1, z: 1 },
});

const helper = app.spawn("Empty", {
  position: { x: 2, y: 0, z: 0 },
});

app.setCameraTarget(cube);

const motion = await app.loadVrm("assets/character.vrm");
app.playVrmMotion(motion, "idle");

input.on("Space", () => {
  helper.transform.rotate(0, 90, 0);
});

What This Covers

Hierarchy Scene View Inspector Game View Script VRM PNG
