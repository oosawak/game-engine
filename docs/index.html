Game Engine

公開用の入口です。実装モックと設計資料をここから開けます。

VRM Editor モーション一覧、名前登録、プレビュー再生をまとめた専用画面のモックです。 VRM Motion Detail モーションごとの Script Name 登録、参照用コード、保存を行う専用画面です。 Sample Page API の使い方と最小の利用例をまとめた入口です。 Editor Mock Cube の追加、Inspector 編集、2D / 3D 切り替え、スクリプト実行を確認できます。 Game View 実行中のシーン、Camera、Light、Object の状態を確認する出力画面です。 Explanation 動画を使って、画面構成と操作の流れを説明するページです。 Shared Assets VRM モーション、PNG、音声、Scene を同一マニフェストで確認するページです。 Repository README リポジトリの概要を確認できます。
