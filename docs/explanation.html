Game Engine / Editor Flow

このページは、`Hierarchy`・`Scene`・`Inspector`・`Project` をどう使うかを短く伝えるための説明ページです。 下の動画は、Cube の追加、選択、移動、スクリプト実行の流れを示しています。

Home Sample Page Open Editor
1. Hierarchy Cube を追加して選択します。選択状態は Scene と Inspector に同期します。
2. Scene View 2D / 3D を切り替えられます。Cube はドラッグで移動できます。
3. Inspector Position / Rotation / Scale とスクリプトを編集できます。編集内容は選択中の Cube に反映されます。
4. Play / Script Play を押すと、各 Cube のスクリプトが実行され、原点周りのループ挙動になります。
5. Sample Page 使い方の例は Sample Page に集約します。Editor を触る前に、API の最小例を確認できます。
